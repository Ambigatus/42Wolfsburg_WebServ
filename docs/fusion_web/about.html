About the project

The aim of the project is to create a web server using C++98 that follows certain standards. This web server is capable of managing various types of HTTP requests, such as GET, HEAD, POST, PUT, and DELETE. It can serve static files from a designated root directory and handle dynamic content using CGI. Importantly, it's designed to handle multiple clients connecting at the same time, and it uses the select() function for this purpose.
